Hi! People call me Tom.

I'm a Principal GIS and Data Platforms Engineer at King County's GIS Center.

Right now I am building a server-side rendered web app to help people find information about the land they own in King County. This app querys data from the county's on-premise databases and leverages hosted map services to tell you everything we can about a parcel number. You can play with the demo here.

This web app is a direct reimplementation of a legacy ASP.NET Webforms project written in Visual Basic .NET and running on .NET Framework 4.5. My replacement is ASP.NET Core Razor Pages project written in C# running on .NET 7. You can read about the performance and maintain ability advantages of this solution here.
